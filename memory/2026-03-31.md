## 项目：梦幻西游手游脚本

### 蛋总的需求
- 目标：制作梦幻西游手游的自动脚本
- 电脑配置：9800X3D + 9070 XT + 32GB DDR5
- 需求：多开（起步5个）、稳定、不被封

### 技术选型决策
1. **模拟器：MuMu（网易）**
   - 雷电是备选（多开更强但特征明显）
   - 理由：默认安卓12更像真机，梦幻西游自家兼容性好
2. **脚本语言：Python**
   - 依赖：OpenCV、numpy、Pillow、ppadb/easyocr
3. **防封策略**
   - 随机点击延迟（0.8-2.5秒）
   - 随机坐标偏移
   - 操作间隔随机发呆
   - 不用固定坐标，用图像识别
4. **ADB连接**
   - MuMu默认端口：16384 + 实例索引
   - 实例0=16384, 实例1=16385, ...

### IP问题
- 5个账号同IP有风险但可接受
- 主要靠脚本行为防封，不是换模拟器
- 先跑5个看效果，被封再考虑IP分散
