## 浏览器控制技术（重要！）

### 方法：用pyppeteer控制系统的Chrome

**安装：**
```bash
unset HTTP_PROXY && unset HTTPS_PROXY && unset ALL_PROXY
pip3 install pyppeteer
```

**使用方法：**
```python
import asyncio
from pyppeteer import launch

async def main():
    browser = await launch(
        executablePath='/Applications/Google Chrome.app/Contents/MacOS/Google Chrome',
        headless=True,
        args=['--no-sandbox', '--disable-setuid-sandbox', '--disable-dev-shm-usage']
    )
    page = await browser.newPage()
    await page.goto('https://artale.app/search-drop', {'waitUntil': 'networkidle0', 'timeout': 60000})
    await asyncio.sleep(2)
    
    # 搜索
    await page.type('input[type="text"]', '搜索词', {'delay': 30})
    await asyncio.sleep(3)
    
    # 获取渲染后的HTML
    html = await page.evaluate('document.body.innerHTML')
    print(html)
    
    await browser.close()

asyncio.get_event_loop().run_until_complete(main())
```

**关键点：**
- Chrome路径：`/Applications/Google Chrome.app/Contents/MacOS/Google Chrome`
- 需要加参数：`--no-sandbox --disable-setuid-sandbox`
- 搜索后要等待JS渲染（约3秒）
- 可以用`page.evaluate()`执行JS获取渲染后的内容

**解决的问题：**
- Cloudflare动态拦截
- JS动态加载的网站（如artale.app）
- 需要登录/验证的页面（通过正常浏览器）
