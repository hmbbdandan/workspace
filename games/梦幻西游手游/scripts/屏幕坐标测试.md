> 用户屏幕：MacBook Pro + 副屏幕(D2)

## 屏幕坐标系统

### 屏幕布局
- **主屏幕 (D1)**：MacBook Pro内置，1512 x 982
- **副屏幕 (D2)**：在主屏幕右侧，游戏运行在此
- **游戏窗口位置**：D2上 (28, 148)，宽326 x 高453
- **游戏窗口全局坐标**：(1512 + 28, 148) = **(1540, 148)**

### 坐标计算公式
```
全局X = 主屏幕宽度(1512) + D2相对X
全局Y = D2相对Y
```

## 脚本开发计划

### 1. 截图模块
```python
import subprocess
def screenshot():
    subprocess.run(['/usr/sbin/screencapture', '-x', '-D', '2', '/tmp/game.png'])
```

### 2. 坐标转换
```python
GAME_OFFSET_X = 1540
GAME_OFFSET_Y = 148
GAME_WIDTH = 326
GAME_HEIGHT = 453

def click_game(rel_x, rel_y):
    """点击游戏窗口内的相对坐标"""
    import pyautogui
    abs_x = GAME_OFFSET_X + rel_x
    abs_y = GAME_OFFSET_Y + rel_y
    pyautogui.click(abs_x, abs_y)
```
