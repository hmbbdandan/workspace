# 梦幻西游手游 - 截图素材指南

> 创建时间：2026-04-02
> 用途：为脚本提供模板匹配的截图素材

---

## 需要截取的素材

请在游戏主界面（不是战斗）时截图，使用 `Alt + PrintScreen`（PrtScn）或模拟器自带截图功能。

---

### 1. 小地图 (minimap.png)
- **位置**：游戏左上角
- **要求**：只截小地图区域，周围少留空白
- **用途**：检测是否在主界面

### 2. 师门师父NPC (master_npc.png)
- **位置**：师门师父角色
- **要求**：只截师父角色，周围少留空白
- **用途**：在游戏场景中定位师父

### 3. 师门师父在大地图上的图标 (master_map_icon.png)
- **位置**：打开地图后，师父的标记点
- **要求**：只截图标，周围少留空白
- **用途**：地图导航

### 4. 对话框 - 师门按钮 (dialog_shimen_btn.png)
- **位置**：与师父对话时，对话框中的"师门"按钮
- **要求**：只截按钮区域
- **用途**：自动接受任务

### 5. 任务追踪栏中的师门任务 (task_tracker.png)
- **位置**：屏幕右侧的任务追踪栏
- **要求**：包含任务描述文字的区域
- **用途**：识别任务状态

### 6. 战斗自动按钮 (battle_auto_btn.png)
- **位置**：战斗界面右下角
- **要求**：只截"自动"按钮
- **用途：一键开启自动战斗

### 7. 捕捉按钮 (catch_btn.png)
- **位置**：战斗胜利后（如果是抓怪任务）
- **要求**：只截"捕捉"选项
- **用途**：抓怪任务完成后捕捉

### 8. 任务完成确认按钮 (reward_claim_btn.png)
- **位置**：交任务后的奖励确认按钮
- **要求**：只截按钮区域
- **用途**：领取奖励

### 9. 送货NPC (delivery_npc.png)
- **位置**：送货任务的目的地NPC
- **要求**：只截NPC角色
- **用途**：定位送货目标

### 10. 商店NPC (shop_npc.png)
- **位置**：杂货店或采购NPC
- **要求**：只截NPC角色
- **用途**：采购任务

## 截图方法

### 方法1：模拟器截图（推荐）
1. 在MuMu模拟器窗口点击右上角菜单
2. 选择"截图"或"屏幕录制"
3. 保存到 `assets/shimen/` 目录

### 方法2：Windows截图
```
Alt + PrintScreen: 截取当前窗口
Win + PrintScreen: 截取全屏并保存到图片
```

### 方法3：ADB截图
```bash
# 连接模拟器
adb connect 127.0.0.1:16384

# 截图
adb -s 127.0.0.1:16384 exec-out screencap -p > screenshot.png

# 然后手动裁剪需要的区域
```

---

## 截图要求

1. **格式**：PNG
2. **分辨率**：与游戏分辨率一致（不要拉伸）
3. **背景**：尽量只截目标元素，周围少留空白
4. **命名**：使用英文名称，见上面列表

---

## 坐标系统说明

游戏内坐标（示例）：
```
游戏窗口分辨率: 960 x 540 (模拟器默认)

小地图位置: (50, 50)
师门师父位置: (150, 200)
对话框师门按钮: (163, 350)
任务追踪栏: (300, 200) - (326, 400)
自动战斗按钮: (280, 400)
```

实际坐标需要根据你的模拟器分辨率进行调整。

## 状态检测

脚本需要检测以下UI状态：

| 状态 | 检测方式 |
|------|----------|
| 主界面 | 小地图存在 |
| 对话框 | 对话框UI出现 |
| 战斗状态 | 敌人血条、行动按钮 |
| 任务追踪 | 右侧任务栏有师门任务 |
| 地图界面 | 大地图UI |
| 奖励弹窗 | 奖励确认按钮 |
