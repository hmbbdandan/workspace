## 游戏概述

梦幻西游手游是网易旗舰IP"梦幻西游"的手游版本，延续了端游的经典玩法，以中国古典名著《西游记》为蓝本。

**游戏特色：**
- 经典回合制战斗
- 丰富社交系统（帮派、结婚、结拜）
- 宠物养成与捕捉
- 自由交易经济系统

## 核心玩法

### 1. 回合制战斗
- 玩家和宠物一起上阵（最多4+1）
- 回合内按速度决定行动顺序
- 技能：物理攻击、法术攻击、封印、治疗、增益/减益
- 自动战斗：可设置自动普通攻击/技能/吃药

### 2. 任务系统
- **主线剧情**：跟随唐僧师徒取经
- **日常任务**：师门任务、宝图任务、捉鬼任务
- **副本**：水陆大会、闯关、秘境降妖等
- **活动**：科举大赛、门派闯关、封妖等

### 3. 宠物（召唤兽）系统
- 可捕捉野生怪物作为宠物
- 宠物有：资质、成长、技能、进阶
- 可以：打书、炼妖、内丹、神兽
- 热门宠物：忘川、幽灵、律法女娲、机关鸟

### 4. 装备与打造
- 装备：武器、衣服、帽子、鞋子、腰带、项链
- 打造需要：书铁材料 + 强化石
- 特效：简易、专注、神佑、反击等
- 简易装备：可突破等级限制

### 5. 社交系统
- 帮派系统（帮派技能、帮战）
- 结婚系统（需双方同意）
- 结拜系统（2-5人）
- 师徒系统

### 6. PVP玩法
- 擂台切磋
- 竞技场（比武大会）
- 帮派战
- 跨服战

## 操作方式

- **战斗**：回合制，自动/手动切换，可设置自动策略
- **移动**：点击地图自动寻路
- **交互**：点击NPC、道具、玩家
- **界面**：底部标签页（任务、宠物、装备、帮派、好友等）

## 自动脚本重点需求分析

根据游戏特性，脚本开发重点：

### 高优先级（日常必做）
1. **师门任务** - 10/20次/天，奖励大量经验+钱
2. **宝图任务** - 30次/天，奖励藏宝图
3. **捉鬼任务** - 100次/天，奖励经验+物品
4. **副本自动战斗** - 水陆大会等

### 中优先级（稳定收益）
5. **自动摆摊交易** - 买卖物品
6. **宠物培养循环** - 捕捉→炼妖→打书
7. **自动组队** - 匹配队友

### 低优先级（特殊场景）
8. **帮派战自动操作**
9. **竞技场自动PK**

---

## 技术注意点

1. **回合制战斗检测** - 识别我方/敌方出手状态
2. **NPC对话框处理** - 确认/取消按钮识别
3. **体力活力系统** - 体力限制每次操作
4. **封禁风险** - 网易有检测机制，需随机化操作
5. **模拟器兼容性** - 雷电/夜神/蓝叠等模拟器
