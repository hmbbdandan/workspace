## 基本信息

- **游戏名称：** 冒险岛 / MapleStory / MapleStory M
- **开发商：** Nexon（韩国）
- **国服代理：** 盛大游戏
- **游戏类型：** 2D横版卷轴MMORPG / 经典休闲MMO
- **核心特色：** 泡泡堂式横版战斗 + 角色养成 + 经济系统

## 游戏特点

- **画面风格：** 2D卡通像素风，色彩鲜艳，角色可爱

- **经济系统：** 自由交易、拍卖行、NFT铸造（冒险岛有区块链道具系统）
- **社交系统：** 公会、组队、结婚系统

## 核心玩法

### 1. 刷级路线（0-100速通）
- **0-30级**：新手村 + 主线任务
- **30-60级**：副本冲刺（妖精学院、黄金海岸）
- **60-100级**：宁静沼泽、天空之城独角兽森林

### 2. 搬砖收益场景
| 场景 | 收益 |
|------|------|
| 宁静沼泽土龙（68-70级） | 单角色日均5-10万金币 |
| BOSS掉落（混沌扎昆/狮子王） | 稀有装备20万+/件 |
| 乌勒斯双倍（晚8-10点） | 单次2000万+金币 |

### 3. 材料与装备
- **普通材料**：灰卵、紫卵（分解装备获取）
- **稀有装备**：潜能装备（洗练+陶德）
- **NFT道具**：限定点装（可铸造上链交易）

### 4. 日常必做
- 每日怪物公园（2次）
- BOSS挑战（扎昆、狮子王等）
- 公会任务

## 自动脚本潜在方向

> ⚠️ 注意：冒险岛国服有检测机制，需参考梦幻西游防封经验

### 适合自动化的任务
1. **日常任务**：怪物公园、每日活动
2. **刷级脚本**：自动寻路 + 自动战斗
3. **材料采集**：特定地图循环刷怪
4. **BOSS挑战**：组队自动配合

### 技术难点
1. **横版2D画面识别**：比梦幻西游简单（2D vs 回合制）
2. **技能释放节奏**：部分职业需要按键时机
3. **地图切换检测**：需要识别传送门/NPC
