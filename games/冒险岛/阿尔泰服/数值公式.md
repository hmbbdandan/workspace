## 一、伤害公式（物理系）

### 基础公式

```
MAX = (主属性 * 系数 + 副属性) * 武器攻击 / 100
MIN = (主属性 * 0.9 * 掌握度 + 副属性) * 武器攻击 / 100
```

### 各武器系数

| 武器类型 | 主属性系数 | 副属性 |
|----------|-----------|--------|
| 单手剑 | STR * 4.0 | DEX |
| 单手斧/魔剑 | STR * 4.4 | DEX |
| 双手剑 | STR * 4.6 | DEX |
| 双手斧 | STR * 4.8 | DEX |
| 长矛（刺） | STR * 5.0 | DEX |
| 矛（挥） | STR * 3.0 | DEX |
| 弓 | DEX * 3.4 | STR |
| 十字弩 | DEX * 3.6 | STR |
| 双刀/飞侠 | LUK * 3.6 | STR+DEX |
| 拳套 | STR * 4.8 | DEX |
| 手枪 | DEX * 3.6 | STR |

## 二、伤害公式（魔法系）

### 法术伤害

```
MAX = ((魔攻²/1000 + 魔攻)/30 + 智力/200) * 魔法攻击
MIN = ((魔攻²/1000 + 魔攻 * 掌握度 * 0.9)/30 + 智力/200) * 魔法攻击
```

### 火毒系特殊

```
毒雾/火焰 每秒伤害 = 怪物HP / (70 - 技能等级)
```

## 三、防御公式

### 物理防御

```
MAX伤害 = 防御前MAX * (1 - 0.01*D) - 物防*0.5
MIN伤害 = 防御前MIN * (1 - 0.01*D) - 物防*0.6

D = 怪物等级 - 角色等级（如果怪物等级 > 角色等级，否则 D=0）
```

### 魔法防御

```
MAX伤害 = 防御前MAX - 魔防*0.5 * (1 + 0.01*D)
MIN伤害 = 防御前MIN - 魔防*0.6 * (1 + 0.01*D)
```

---

## 四、命中率公式

```
命中率 = 命中 / ((1.84 + 0.07 * D) * 回避) - 1

D = 怪物等级 - 角色等级
```

## 六、关键概念

- **主属性**：STR（力量）、DEX（敏捷）、INT（智力）、LUK（运气）
- **副属性**：辅助主属性的属性
- **掌握度**：各职业特有的技能加成（如剑士Mastery、射手Expert等）
- **武器系数**：不同武器有不同的伤害倍率
- **元素优势/劣势**：火冰雷互克，优势造成150%伤害，劣势造成50%伤害
