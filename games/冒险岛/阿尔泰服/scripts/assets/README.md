## 截图方法

### 方法1：游戏内截图
1. 打开游戏设置
2. 找到"截图"或"录像"选项
3. 保存到指定文件夹

### 方法2：模拟器截图
MuMu模拟器：
- 窗口右上角 → 截图按钮
- 或 `Ctrl + Shift + S`

### 方法3：Windows截图
```
Alt + PrintScreen: 截取当前窗口
Win + PrintScreen: 截取全屏
```

### 方法4：ADB截图
```bash
# 连接模拟器
adb connect 127.0.0.1:16384

# 截图并pull到电脑
adb exec-out screencap -p > screenshot.png

# 然后用图片编辑器裁剪
```

## 截图要求

1. **格式**：PNG（不要用JPG）
2. **分辨率**：与游戏分辨率一致（MuMu默认960x540）
3. **背景**：尽量只截目标元素，周围少留空白
4. **命名**：使用英文名称，见上面列表
5. **尺寸**：长宽在20-100像素之间效果最好

---

## 模板匹配说明

脚本使用 OpenCV 的 `matchTemplate` 函数进行模板匹配。

- 阈值建议：`0.75 - 0.85`
- 阈值太低会误匹配
- 阈值太高会漏匹配

匹配结果说明：
- `confidence > 0.9`: 几乎肯定
- `confidence > 0.8`: 很可能是
- `confidence > 0.75`: 可能是

## 调试技巧

如果脚本找不到目标：

1. **检查模板是否存在**
   ```bash
   ls assets/monsters/*.png
   ```

2. **测试模板匹配**
   - 把截图放到 `test_screenshot.png`
   - 手动运行匹配代码看结果

3. **调整阈值**
   - 编辑 `config.py`
   - 降低 `template_match_threshold`

4. **检查坐标**
   - 确认游戏窗口位置
   - 确认ADB截图的分辨率
