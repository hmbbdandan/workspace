## 拍卖行系统

### 官方拍卖行
- 2025年6月12日更新后正式开放
- 支持道具买卖、搜索
- ⚠️ 30级以前无法使用拍卖行

### 第三方交易平台
由于官方拍卖行曾长时间维护，出现了替代方案：
- **Artale Market** — 玩家开发的第三方交易平台，与Discord绑定
- 用于道具上架、查询、交易

### 交易注意事项
- 拍卖行有价格波动，善用NPC商店与拍卖行价差
- 避免当"盘子"（花冤枉钱）

## ⚠️ 封号风险（重要）

> 百度贴吧信息：阿尔泰封号规则

| 违规行为 | 处罚 |
|----------|------|
| 使用外挂/脚本 | 1天至永久封号 |
| 自动打怪 | 检测 |
| 修改游戏数据 | 永久封号 |

**检测方式：**
- 实时检测（抓现行）
- 离线追封（事后追责）

**结论：脚本有明确封号风险**，开发需谨慎考虑防封策略

## 脚本开发技术方案（参考）

基于现有开源项目 **MapleStoryAutoLevelUp**：
- 技术：计算机视觉（OpenCV）识别游戏画面
- 不修改游戏内存，纯屏幕检测
- 识别血条、怪物位置
- 模拟键盘命令控制角色

**与梦幻西游脚本的差异：**
- 冒险岛是横版2D，动作类游戏
- 需要识别怪物位置+角色状态
- 技能释放时机更关键
- 防封需求更高

## 游戏玩法与特性

### 经济系统

**货币类型：**
- **枫币（Mesos）** — 游戏内主要货币
- **世界币（World Coins）** — 氪金货币，用于商城购买

**交易方式：**
1. **拍卖行** — 官方道具交易系统（2025年6月重新开放）
2. **Artale Market** — 玩家自建第三方交易平台，与Discord绑定
3. **玩家直传** — 面对面交易

**物价参考（2025年中）：**
- 雪花：1000个 ≈ 2350T（台币）
- 枫币：1E ≈ 650T
- 棒冰棒：12000个 = 200T
- 矿泉水：30000个 = 350T

**金币赚取途径：**
- 打怪掉落材料/装备
- 任务奖励
- 副本通关奖励
- 卷轴/飞镖制作贩卖
- BOSS掉落

### 装备与强化

**装备来源：**
- 打怪掉落
- NPC商店购买
- 拍卖行购买
- 制作获得

**强化方式：**
- **卷轴冲装备** — 成功率不等，失败可能掉级
- **潜能/星之力** — 额外属性加成
- **装备升级** — 低级装备进阶

**省成本技巧：**
- NPC商店与拍卖行价格差
- 用NPC便宜货冲，省拍卖行贵货

### 组队系统

**组队副本（5个）：**

| 副本 | 等级 | 难度 | 奖励 |
|------|------|------|------|
| 月妙 | LV10+ | 简单 | 经验、道具 |
| 废弃下水道 | LV21+ | 简单 | 经验、装备 |
| 天空之城女神塔 | LV51+ | 中等 | 经验、稀有装备 |
| 武陵海盗 | LV55+ | 中等 | 经验、装备 |
| 罗密欧与朱丽叶 | LV71+ | 困难 | 大量经验、稀有装备 |

**组队机制：**
- 4人组队副本
- 经验共享（队友获得额外经验加成）
- 20%BOSS掉落分配机制

### BOSS系统

**野外BOSS：**
- 全岛17只野外BOSS
- 有固定刷新时间
-掉落稀有装备和大量金币

**BOSS计时器：**
- http://artaletool.com/boos（玩家自建工具）

### NFT/区块链系统

**特殊道具：**
- 部分道具可铸造为NFT
- NFT道具可在区块链上交易
- 稀有限定道具价值较高

**铸造流程：**
道具栏 → MINTING → 铸造后上架交易所

### 升级特性

**Artale vs 传统冒险岛：**
- 蜗牛壳不消耗（初期省钱）
- 恢复技能改为持续回血（省药水）
- 战士HP恢复技能增强

### 脚本框架（已搭建）

- 路径：`阿尔泰服/scripts/`
- 架构：screen_capture + keyboard_controller + auto_levelup
- 依赖：OpenCV, pynput, pyautogui
- 状态：框架完成，待截图素材配置

### 数据分析

**发现：**
- 主教（牧师）单人刷怪效率明显低于输出职业
- 高等级主教在团队中价值高（加血加盾）
- 牧师在"漂浮幽灵墓地"等幽灵系地图有优势（圣光克制）

**升级效率对比（真实数据）：**
- 黑骑士/冰雷：1.5M-2M EXP/h（高级地图）
- 主教：50-80万 EXP/h（单刷）
- 差距约3-4倍

**但牧师的独特价值：**
1. 回血技能省药水（实际药水消耗接近零）
2. 组队时是核心辅助
3. 幽灵系地图掉落值钱
